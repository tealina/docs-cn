# Overview

Tealina 需要一些基本的配置, 使用 create-tealina 创建一个开箱即用的脚手架项目.

## 创建脚手架项目

::: tip Node 版本
Tealina 需要你的 [Node.js](https://nodejs.org/en/) 版本是 18+. 20+.
:::

::: code-group

```bash [PNPM]
$ pnpm create tealina

```

```bash [NPM]
$ npm create tealina@latest
```

```bash [Yarn]
$ yarn create tealina
```

```bash [Bun]
$ bun create tealina
```
:::

跟随提示即可。
